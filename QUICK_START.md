# Quick Start (2 minutes)

## Install & Login

```bash
npx overseer login
```

Browser opens, you log in. Done.

## Start Watching

```bash
npx overseer watch .
```

Dashboard opens automatically. You see real-time analysis as you code.

## That's it

- Write code
- Overseer analyzes
- Dashboard shows feedback
- You iterate

## Commands

```bash
npx overseer login          # Login (one time)
npx overseer watch .        # Watch current directory
npx overseer watch ./src    # Watch specific folder
npx overseer --help         # Show all commands
```

## Project Context (optional)

Add these files to your project for better analysis:

- `ARCHITECTURE.md` — Design decisions
- `TECH_CHOICES.md` — Why each tech
- `SECURITY.md` — Security requirements
